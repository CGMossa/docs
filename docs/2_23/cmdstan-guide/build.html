2.2 Building the CmdStan Tools

The CmdStan makefile target build sets up the CmdStan utilities and compiles all necessary C++ libraries.

> make build

This will build utilities stansummary and diagnose and instantiate the stanc binary. If your computer has multiple cores and sufficient ram, the build process can be parallelized by providing the -j option. For example, to build on 4 cores, type:

> make -j4 build

Warning: The make program may take 10+ minutes and consume 2+ GB of memory to build CmdStan.
